HOW THE INTERNET ACTUALLY WORKS?

What is the Internet?

The internet is simply a huge network of computers connected all around the world. You can think of it like a giant spider’s web where millions of devices are talking to each other. Because of this network, we can send emails, watch videos, chat on WhatsApp, and browse websites.

How does the Internet actually work?
  1. Your computer first asks for the IP address of “google.com” (since computers only understand numbers, not names).
  2. A DNS server (like a phonebook) gives the correct IP address.
  3. Your internet provider (Jio, Airtel, etc.) carries your request forward.
  4. Google’s server sends back the web page files.
  5. Your browser reads those files and displays the Google page you see.
  6. So, every time you open a website, it’s basically a conversation between your computer (client) and another computer (server).

The ipconfig command

On Windows, if you type ipconfig in the Command Prompt, it shows details of your computer’s network: your IP address, your gateway, etc. This is often used to troubleshoot network problems.

What is a Browser?

A browser is simply the software you use to access the web (Chrome, Firefox, Edge, Safari). Its job is to understand web files (HTML, CSS, JavaScript) and display them as a nice-looking webpage.

What is a Domain?

A domain is the human-friendly name of a website. Instead of remembering 142.250.67.14, you just type google.com. That’s the beauty of domains.

What happens when you open a website?

  1. Your browser sends a request to the server.
  2. The server replies with files (HTML, CSS, JS, images).
  3. The browser puts all these pieces together and shows the final webpage.

  • CSS = design/look
